# Jira Teams

## A Jira Clone

Jira Teams is a cost-effective and feature-rich solution as an alternative for the project management software Jira.

## Dev Guidelines

### Installing all apps

```bash
pnpm install
```

### Running the `api` app

```bash
pnpm api dev
```

### Running the `web` app

```bash
pnpm web dev
```

### Running all apps at once

```bash
pnpm dev
```

### Adding Components to @repo/ui

```bash
pnpm ui:add <component_1> <component_2>
```

### Source Code Formatting

```bash
pnpm format
```

### Types checking

```bash
pnpm check-types
```

### Building the app

```bash
pnpm build
```

or use shorthand filters to build individually. Example below.

```bash
pnpm web build
```

### Running the Lint

```bash
pnpm lint
```

### Commiting changes to git

```bash
pnpm commit
```
